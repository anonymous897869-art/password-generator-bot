# 🔐 Password Generator Bot

A Python-based tool that generates secure random passwords using letters, numbers, and symbols.

---

## 🚀 Features
- Generates strong random passwords
- User chooses password length
- Uses letters, digits, and symbols
- Beginner-friendly Python project

---

## 🛠️ Tech Stack
- Python 3
- Random module
- String module

---

## ⚙️ How It Works
1. Takes password length as input  
2. Combines letters, numbers, and symbols  
3. Randomly selects characters  
4. Displays generated password
